  exported/
    1x/               ← 1倍图（标准清晰度）
    2x/               ← 2倍图（Retina / iPhone 推荐）

🎬 呼吸灯动画使用方式

以下图标需要配合 CSS 动画实现呼吸灯效果（PNG 本身不含动效）：

  • ai-companion — 天线球呼吸（2秒周期）
  • plan-calendar — 日历光晕脉冲（2秒周期）
  • mic-recording — 麦克风脉冲（1.5秒周期）
  • poem-sparkle — 星星闪烁（1.5秒周期）
  • tab-ai-diary — AI日记 Tab 天线（2秒周期）

在 icon-animations.wxss 中复制对应的 @keyframes 和 .icon-anim-* class 到你的 WXSS 文件即可。
