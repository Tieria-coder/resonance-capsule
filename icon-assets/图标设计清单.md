## 六、文件对照表

替换图标时，需要修改以下文件中的 emoji：

| 文件 | 说明 |
|------|------|
| `utils/util.js` | 情绪大类 + 子情绪 + 时间段定义 |
| `pages/index/index.wxml` | UI功能图标（AI、聊天、录音、关闭等） |
| `pages/plan/plan.wxml` | 计划页图标 |
| `pages/report/report.wxml` | 报告页图标 |
| `pages/calendar/calendar.wxml` | 日历页（暂无emoji） |

### 替换方式

将 emoji 替换为 `<image src="/icons/xxx.png" class="icon-xxx" />` 或自定义 `<icon name="xxx" />` 组件。
