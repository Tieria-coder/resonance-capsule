# 情绪计划功能 - 数据库配置

## 需要创建的集合

在微信开发者工具云开发控制台，创建以下集合：

### 1. emotion_plans（情绪计划集合）

**权限设置：** 仅创建者可读写

**字段说明：**
```json
{
  "user_id": "用户ID",
  "openid": "微信openid",
  "cycle_days": "周期天数（7/14/21）",
  "reminders": ["morning", "evening"],
  "daily_target": "每日目标次数（1-5）",
  "start_date": "开始日期",
  "end_date": "结束日期",
  "status": "状态（active/completed/ended）",
  "completed_days": "已完成天数",
  "flowers": "获得花朵数",
  "created_at": "创建时间",
  "updated_at": "更新时间"
}
```

### 2. users 集合需要更新的字段

在现有 users 集合的 stats 字段中新增：
```json
{
  "stats": {
    "streak": 0,
    "max_streak": 0,
    "total_records": 0,
    "total_flowers": 0  // 新增
  }
}
```

## 功能说明

### 情绪计划是什么？
- 帮助用户养成定时记录情绪的习惯
- 不是控制情绪，只是激励记录
- 7天/14天/21天周期可选

### 核心功能
1. **设定提醒时段** - 上午/午间/下午/傍晚/晚上
2. **每日打卡目标** - 每天记录几次算完成
3. **进度追踪** - 周进度条、每日完成状态
4. **小红花激励** - 完成一天获得一朵花
5. **连续打卡** - 历史计划连续完成周数

### 使用流程
1. 首页点击「情绪打卡计划」入口
2. 选择周期（7/14/21天）
3. 选择提醒时段（可多选）
4. 设定每日目标（1-5次）
5. 开启计划后每天记录情绪
6. 周末查看完成情况，获得小红花
