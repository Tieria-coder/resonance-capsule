### 质感效果

1. **毛玻璃**
   ```css
   backdrop-filter: blur(24rpx) saturate(180%);\n   ```

2. **多层阴影**
   ```css\n   box-shadow:\n     0 8rpx 32rpx rgba(0,0,0,0.12),     /* 主阴影 */\n     0 2rpx 8rpx rgba(0,0,0,0.08),      /* 近景阴影 */\n     0 0 0 1rpx rgba(255,255,255,0.1) inset, /* 内发光 */\n     0 0 60rpx rgba(120,80,200,0.08);     /* 彩色光晕 */\n   ```

3. **顶部高光** - 左右半圆都有白色渐变高光

4. **微妙纹理** - 中间区域有紫色径向光

5. **悬浮动画** - 缓慢上下浮动（4s 循环）

6. **情绪气泡** - 渐变背景 + 细边框 + 微阴影
