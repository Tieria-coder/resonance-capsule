# 情绪胶囊 - 云数据库集合创建说明

## 需要创建的集合

### 1. `plans` - 情绪计划集合

**字段说明：**
```javascript
{
  "_id": "自动生成",
  "user_id": "用户ID（字符串）",
  "title": "计划标题（字符串）",
  "description": "计划描述（字符串，可选）",
  "emotion_goal": "目标情绪（字符串，如 'happy'）",
  "actions": [
    {
      "text": "行动描述（字符串）",
      "completed": false,  // 是否完成（布尔值）
      "completed_at": null  // 完成时间（日期，可选）
    }
  ],
  "created_at": "创建时间（日期，自动生成）",
  "updated_at": "更新时间（日期，自动生成）"
}
```

**创建方式：**
1. 打开微信开发者工具
2. 点击左侧「云开发」→「数据库」
3. 点击「+」创建集合
4. 集合名称输入 `plans`
5. 点击「确定」

**权限设置：**
- 集合权限设置为「仅创建者可读写」

---

## 自动创建集合（可选）

如果你希望通过代码自动创建集合，可以在 `app.js` 的 `onLaunch` 中添加：

```javascript
// app.js
onLaunch() {
  // 初始化云开发
  if (wx.cloud) {
    wx.cloud.init({
      env: 'cloud1-d5gev82b5db8c2485',
      traceUser: true,
    })
  }
  
  // 自动创建集合（仅首次运行）
  this.initCollections()
  
  // 检查登录状态
  this.checkLogin()
},

async initCollections() {
  const db = wx.cloud.database()
  const collections = ['plans', 'reports']
  
  for (const name of collections) {
    try {
      await db.createCollection(name)
      console.log(`集合 ${name} 创建成功`)
    } catch (err) {
      // 集合可能已存在，忽略错误
      if (err.message && err.message.indexOf('exist') === -1) {
        console.error(`集合 ${name} 创建失败:`, err)
      }
    }
  }
},
```

**注意：** `db.createCollection()` 需要管理员权限，建议在首次部署时手动创建集合。

## 验证集合创建成功

创建完成后，可以通过以下方式验证：

1. 在微信开发者工具中，点击「云开发」→「数据库」
2. 查看左侧集合列表，确认 `plans` 和 `reports` 已创建
3. 在 `pages/plan/plan.js` 中添加测试代码：
   ```javascript
   onLoad() {
     this.testAddPlan()
   },
   
   async testAddPlan() {
     try {
       const db = wx.cloud.database()
       const result = await db.collection('plans').add({
         data: {
           user_id: 'test',
           title: '测试计划',
           description: '这是一个测试计划',
           emotion_goal: 'happy',
           actions: [
             { text: '行动1', completed: false, completed_at: null }
           ],
           created_at: new Date(),
           updated_at: new Date(),
         }
       })
       console.log('测试计划添加成功:', result._id)
     } catch (err) {
       console.error('测试计划添加失败:', err)
     }
   },
   ```
4. 编译运行，查看控制台是否输出成功信息
